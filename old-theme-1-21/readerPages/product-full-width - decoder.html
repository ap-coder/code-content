CortexDecoder Embedded OEM Software SDK

Embedded barcode software development kits for OEM applications and products requiring advanced barcode decoding and machine vision for retail automation, warehouse automation, manufacturing, and other enterprise processes.

  • Features
  • Technical Specifications

CortexDecoder Embedded SDK is for OEM device manufacturers looking to integrate the best, enterprise-grade barcode scanning functionality within the devices they produce.

Purpose built functionality that is truly flexible Whether your product is a purpose-built barcode reader, scan engine, kiosk, mobile computer, or any other device with a camera, CortexDecoder Embedded has been optimized for enterprise-grade performance to overcome the scanning challenges of barcode reading and data capture found throughout many industries.   • Flexible OS Portability

    A small footprint, optimized portability, and decades of experience make for easy integration to virtually any processing platform and operating system.

  • Superior on Poor Quality Barcodes

    Performance beyond the expected. Simply put, we just read barcodes others can’t. Fast, accurate, dependable.

Minimum System Requirements:

  • 500 KB storage
  • 500 KB RAM minimum, 1 MB RAM recommended
  • 1 megapixel image resolution

O/S Support:

Software portability to any processing platform and OS – Android, iOS, Windows, Linux, custom, etc.

Symbologies Supported:

  • Direct Part Marks:
    DPM barcodes applied by laser or chemical etching, or inkjet print. Will also read Data Matrix marks that have been dot peened to a surface.
  • 1D:
    Codabar, Code 11, Code 32, Code 39, Code 93, Code 128, IATA 2 of 5, Interleaved 2 of 5, GS1 DataBar (RSS), Hong Kong 2 of 5, Matrix 2 of 5, MSI Plessey, NEC 2 of 5, Plessey, Straight 2 of 5, Telepen, Trioptic, UPC/EAN/JAN, IATA 2 of 5, NEC 2 of 5, BC412, Pharmacode.
  • Stacked:
    GS1 Composite (CC-A/CC-B/CC-C), MicroPDF417, PDF417, Code 49, Codablock F.
  • 2D:
    Aztec Code, Code 49, Codablock F, Data Matrix, Han Xin, MaxiCode, MicroQR, QR Code, QR Code Model 1, Dot Code, Grid Matrix, GS1 DataBar.
  • Postal Codes:
    USPS Postnet, USPS Planet, Australian Post, USPS Intelligent Mail, Japan Post, KIX Code, UK Royal Mail, Dutch Post, Korea Post, Canada Post, UPU ID-tags.
  • Proprietary 2D:
    GoCode® (Additional License Required).
